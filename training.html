C1 An introduction to XML and eXistdb

XML is everywhere. Since its introduction it has spread into nearly all areas of information technology. Its major advantages are platform-independence, human- and machine-readability and the fact that it is very easy to process and tools are available on every platform.

While rarely being obvious to users XML is a key technology today. XML lets heterogenous systems exchange data and encode any language on earth with a minimum of effort. As such XML it is often called the lingua franca of IT.

This course will introduce into the fundamentals of XML covering the core concepts, available tools for processing and give examples of its applicability.

Goals

  • Refresh your knowledge of XML fundamentals
  • Get to know the tools and interfaces by eXistdb
  • Enable you to write basic queries and set up eXistdb to work efficiently

Target Audience

This course is targeted at system architects and developers that seek a deeper understanding of this fundamental technology. It will provide the fundamental know-how needed to follow and understand the more advanced courses on XPath, XQuery, XSLT and XForms.

Language

This training course is held in English or German

Prerequisites

None

Duration

1 day

C2 XQuery for Developers

XQuery is a Turing-complete functional language for querying and manipulating large amounts of XML. Like XML XQuery is an open standard defined by the W3C. The current language version is 3.0. The course will give an introduction into the fundamental language features accompanied by practical exercises. Step by step the participants will built an example application that has especially be designed for teaching purposes.

Goals

  • Enable participants to write their first XML-based web-applications in XQuery
  • Starting from a given data set, participants learn all the building blocks necessary to create complex applications in XPath and XQuery
  • Enable participants to write their first web interfaces entirely in XQuery

Target Audience

This course is designed for architects and developers starting to work with large volumes of XML or plan to use a XML database. It will cover the basic language concepts and enable the participants to write their own scripts building a complete XML-based application.

Language

This training course is held in English or German

Prerequisites

Participants must have completed the XML Fundamentals course

Duration

2 days

Price (excluding VAT)

790 EUR*

C3 Application Development with eXistdb

eXistdb is a feature-complete platform for developing complete XML-based applications that can be packaged and deployed as a single application archive. Participants will learn how to efficiently develop applications with the builtin tools, package applications, use indexes to speed up searches and process user input.

Goals

  • Create visually appealing web applications using eXistdb's HTML templating
  • Learn about URL rewriting and other features
  • Make the resulting application efficient by creating indexes, profiling queries and testing them

Target Audience

This course targets developers that like to learn about application development with eXistdb. Familiarity with XML, XPath and XQuery is assumed and the course will delve into the eXistdb-specific functionalities and tools that effectively drive app development.

Language

This training course is held in English or German

Prerequisites

Participants must have completed the XQuery for Developers course

Duration

2 days

Price (excluding VAT)

790 EUR*

C4 XForms

XForms is a W3C standard which was designed to be the next generation of form processing and was meant to replace HTML forms. Due to lacking support by browser vendors XForms couldn't spread as intended. This course answers the question why XForms is still relevant and useful especially when used on the server. Participants will learn why XForms is still the most powerful forms framework around and how they can profit from it.

Goals

  • Provide an overview about XForms features
  • Enable participants to write validations and calculations
  • Learn some techniques for building advanced interactions

Target Audience

The audience for this course are architects and developers looking for advanced form capabilities and seeking to understand what XForms can do.

Language

This training course is held in English or German

Prerequisites

Participants must have completed the XML Fundamentals course

Duration

2 days

Price (excluding VAT)

790 EUR*
